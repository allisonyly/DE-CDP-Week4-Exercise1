### Prompts Used in Copilot:
- How can I check if Java and the JDK is installed, I downloaded both from the web but I am still getting a javac compile error
- Help me create an abstract class called Video. The private field names will be: title, genre, available. - Help me create a subclass of Video, called Movie. It will implement play() with a message to the user and there will be an overloaded play() method as well. This is an example that I want for the overloaded play() method: play(String quality) (HD, 4K, etc.)
- Please create another subclass of Video, called Series, where it will implement play() differently such that it will print "Playing <episode num> of <series name>" instead
- Create the main class now, called GoldenVideoStoreApp. In this class, it will have an array of Video objects, which will contain a mix of Movie and Series. Please implement this class so that it can demonstrate: adding videos to the array, looping through and calling play() on each to show polymorphism, using the overloaded play() method, and it will print a list of available videos using an array traversal.
